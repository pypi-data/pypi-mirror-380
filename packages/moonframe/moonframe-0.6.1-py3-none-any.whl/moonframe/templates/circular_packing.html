Worst performers

This is an interactive circular packing graph developed at CERFACS in order to explore the nested hierarchical organization of complex codes.

How to use it

Each circle represents a code component, such as a file, function, or subroutine.

Circle sizes are proportional to the number of lines in the code. Clicking on a circle will zoom in on it.

If you hold Ctrl or ⌘ while clicking a circle, it will open the corresponding code file in VSCode (if you have it).

For more, keep a look at the full documentation.

Support
=======

If you meet any issue or have any feature request, please send it to the support address `support@pitchmeld.ing <support@pitchmeld.ing>`_.

If you meet a bug, please join to your email:

* A minimal code snippet that reproduces the issue.

* The output of the following commands:

.. code-block:: shell

    export PITCHMELD_KEY=api__valid_or_invalid_license_token_
    python -c 'import pitchmeld; print(pitchmeld.info)'

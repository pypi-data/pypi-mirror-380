This module allows you to display the option to print bills from the POS
even if it is not a restaurant type.

**Table of contents**

.. contents::
   :local:

Configuration
=============

To configure this module, you need to:

1. Go to Point of Sale module.
2. Select the Point of Sale -> Settings.
3. Checks 'Early Receipt Printing'.

Usage
=====

To use this module, you need to:

1. Open a new session of the configured point of sale.
2. You will see that you can print receipts before completing the sale.
